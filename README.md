# Hybrid Fake News Detection using TF-IDF, BERT, and Random Forest

This project proposes a hybrid fake news detection framework combining TF-IDF features, BERT embeddings, feature fusion, and Random Forest classification.

Dataset:
- WELFake Dataset

Performance:
- Accuracy: 87.4%
- Precision: 85.66%
- Recall: 90.94%
- F1-Score: 88.22%
- AUC: 94.67%

This Streamlit application demonstrates the deployment interface of the proposed fake news detection framework.

The full experimental pipeline, including TF-IDF feature extraction, BERT embeddings, feature fusion, and Random Forest classification, is implemented in the accompanying notebook.
